# SOLODASH
O SoloDash é uma vitrine inteligente para o músico Joelson Ramos. Criada como SPA nativa de alta performance, a plataforma automatiza propostas via SoloAI, protege negociações com Chat Seguro (filtros Regex) e centraliza a gestão de agenda e repertório, profissionalizando a carreira do artista de forma ágil, tecnológica e segura.
